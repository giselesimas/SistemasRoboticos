Instalação do ROS Melodic com Gazebo, RViz e TurtleBot3 no Ubuntu 18.04

Objetivo do Material

Instalar o ROS Melodic em uma máquina com Ubuntu 18.04 e configurar o Gazebo, RViz e os pacotes do TurtleBot3 para desenvolvimento robótico.

Pré-requisitos

  • Ubuntu 18.04 instalado.
  • Conexão com a internet.
  • 2 GB de espaço livre em disco (mínimo).

Instalação do ROS Melodic

Configurar os Repositórios ROS

sudo apt update
sudo apt install curl gnupg2 lsb-release
curl -sSL 'http://repo.ros2.org/repos.key' | sudo apt-key add -
sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'
sudo apt update

Instalar o ROS Melodic

sudo apt install ros-melodic-desktop-full
sudo rosdep init
rosdep update

Configurar o Ambiente ROS

echo "source /opt/ros/melodic/setup.bash" >> ~/.bashrc
source ~/.bashrc

Instalação do Gazebo e RViz

Instalando o Gazebo

sudo apt install ros-melodic-gazebo-ros-pkgs ros-melodic-gazebo-ros-control

Instalando o RViz

sudo apt install ros-melodic-rviz

Instalação dos Pacotes do TurtleBot3

sudo apt install ros-melodic-turtlebot3 ros-melodic-turtlebot3-simulations
echo "export TURTLEBOT3_MODEL=burger" >> ~/.bashrc
source ~/.bashrc

Executando o TurtleBot3 no Gazebo

roscore
roslaunch turtlebot3_gazebo turtlebot3_empty_world.launch

Usando o RViz para Visualizar o TurtleBot3

roslaunch turtlebot3_bringup turtlebot3_model.launch

Controlando o TurtleBot3 na Simulação

Controle pelo Teclado

roslaunch turtlebot3_teleop turtlebot3_teleop_key.launch

Publicação de Comandos de Movimento

rostopic pub /cmd_vel geometry_msgs/Twist '{linear: {x: 0.2, y: 0.0, z: 0.0}, angular: {x: 0.0, y: 0.0, z: 0.5}}'

Testando Sensores do TurtleBot3

rostopic echo /scan

Configuração do Workspace ROS (opcional)

mkdir -p ~/catkin_ws/src
cd ~/catkin_ws/
catkin_make
echo "source ~/catkin_ws/devel/setup.bash" >> ~/.bashrc
source ~/.bashrc

Resumo e Considerações

Agora você possui uma instalação completa do ROS Melodic no Ubuntu 18.04 com Gazebo, RViz e os pacotes do TurtleBot3. Esse ambiente permite desenvolver e testar algoritmos de controle e navegação com o TurtleBot3 em simulação.
